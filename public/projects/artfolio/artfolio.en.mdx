## Project Overview

GitHub: [https://github.com/leey00nsu/ArtFolio-FE](https://github.com/leey00nsu/ArtFolio-FE)

ArtFolio is an AI-based art auction service.
It analyzes images uploaded by users using AI and automatically generates descriptions and voice narration tailored to the artwork.
The project was carried out with 2 frontend developers, 2 backend developers, and 1 designer; I served as the frontend lead and team leader.
We received an honorable mention at the Sahmyook University SW Project Competition.

## Key Features

- Real-time art auction listings
- Real-time auction price charts
- AI-powered artwork evaluation charts

## Tech Stack

### Frontend

- Framework: React
- Language: TypeScript
- Styling/UI: Tailwind CSS
- State management: React Query (TanStack Query), Zustand
- Charts: Nivo
