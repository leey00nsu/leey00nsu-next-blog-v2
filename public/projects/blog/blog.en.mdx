## Project Overview

Github: [https://github.com/leey00nsu/leey00nsu-next-blog-v2](https://github.com/leey00nsu/leey00nsu-next-blog-v2)

This is a Next.js MDX-based technical blog that uses an in-house editor to create posts, leverages AI to generate Korean/English multilingual documents, and can automatically commit them to GitHub branches.

## Key Features

- Content authoring via a web editor based on MDX files
- Generate multilingual documents using AI
- Automatic commits to GitHub branches
- LCP optimization by generating image metadata
- Applied FSD (Feature-Sliced Design) structure for project modularization

## Tech Stack

- Framework: Next.js
- Language: TypeScript
- Styling/UI: Tailwind CSS, shadcn/ui
- MDX: next-mdx-remote, remark-gfm, remark-breaks, rehype-slug, rehype-pretty-code
- Internationalization: next-intl
- Editor: @mdxeditor/editor
- Authentication: next-auth@5 (GitHub Provider)
- Translation: OpenAI SDK
- Auto-commit: Octokit (GitHub API)
- Image processing: sharp, lqip-modern
- Utilities: gray-matter, zod, es-toolkit, lucide-react, react-hook-form
- Dev tools: ESLint, Prettier, Husky, lint-staged
