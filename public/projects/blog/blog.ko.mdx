## 프로젝트 소개

Github 주소 : [https://github.com/leey00nsu/leey00nsu-next-blog-v2](https://github.com/leey00nsu/leey00nsu-next-blog-v2)

Next.js MDX 기반의 기술 블로그로 자체 에디터에서 작성한 글을 AI를 활용해 한국어/영어 다국어 문서를 생성하고, GitHub 브랜치로 자동으로 커밋할 수 있습니다.

## 주요 기능

- mdx 파일을 기반으로한 웹 에디터를 통한 콘텐츠 작성
- AI를 활용한 다국어 문서 생성
- GitHub 브랜치로 자동 커밋
- 이미지 메타데이터 생성을 통해 LCP 최적화
- 프로젝트 모듈화를 위한 FSD(FEature-Sliced Design) 구조 적용

## 기술 스택

- 프레임워크: Next.js
- 언어: TypeScript
- 스타일/UI: Tailwind CSS, shadcn/ui
- MDX: next-mdx-remote, remark-gfm, remark-breaks, rehype-slug, rehype-pretty-code
- i18n: next-intl
- 에디터: @mdxeditor/editor
- 인증: next-auth@5(GitHub Provider)
- 번역: OpenAI SDK
- 자동 커밋: Octokit(GitHub API)
- 이미지 처리: sharp, lqip-modern
- 유틸: gray-matter, zod, es-toolkit, lucide-react, react-hook-form
- 개발 도구: ESLint, Prettier, Husky, lint-staged
