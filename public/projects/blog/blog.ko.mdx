## 프로젝트 소개

GitHub : [https://github.com/leey00nsu/leey00nsu-next-blog-v2](https://github.com/leey00nsu/leey00nsu-next-blog-v2)

Next.js 기반의 기술 블로그입니다. 자체 웹 에디터에서 작성한 글을 AI가 한국어/영어로 자동 번역하고, GitHub API를 통해 브랜치에 직접 커밋하여 배포까지 자동화했습니다.
**글쓰기부터 배포까지 완전 자동화된 블로그 파이프라인**을 구축한 것이 특징입니다.

## 주요 기능

- 웹 에디터를 통한 MDX 문서 작성
- AI 자동 번역(한국어 ↔ 영어) 지원
- GitHub API 연동 자동 커밋
- 이미지 메타데이터 추출 및 LCP/CLS 개선
- FSD 아키텍처 기반 모듈화

## 기술 스택

- Frontend: Next.js, TypeScript, Tailwind CSS
- Infra: GitHub API (Octokit), OpenAI SDK
