## Project Overview

GitHub : [https://github.com/leey00nsu/leemage](https://github.com/leey00nsu/leemage)

This is a self-hosted image management platform using Oracle Cloud Infrastructure (OCI) Object Storage.  
It implements features similar to Cloudinary, enabling efficient project-level image management and providing developer-friendly APIs.

## Key Features

- Image upload and management based on OCI Object Storage
- Project-scoped image management and APIs
- Responsive dashboard and intuitive UI
- Scalability through FSD (Feature-Sliced Design) architecture

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: PostgreSQL, Prisma, Next.js API Routes
- Infra: Oracle Cloud Infrastructure (Object Storage)
