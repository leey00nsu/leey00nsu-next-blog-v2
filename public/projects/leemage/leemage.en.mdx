## Project Overview

GitHub: [https://github.com/leey00nsu/leemage](https://github.com/leey00nsu/leemage)

Leemage is an image management platform built using Oracle Cloud Infrastructure (OCI) Object Storage. It provides features similar to Cloudinary and enables efficient image management on a per-project basis. Using Next.js and modern web technologies, it offers an intuitive, responsive user interface and developer-friendly APIs for easy integration with external services.

## Key Features

- Image management using Oracle Cloud Infrastructure (OCI) Object Storage
- Project-based image organization
- FSD (Feature-Sliced Design) structure applied for project modularization

## Tech Stack

### Frontend

- Framework: Next.js
- Language: TypeScript
- Styling/UI: Tailwind CSS, Shadcn
- State management: React Query (TanStack Query)

### Backend

- Framework: Next.js API routes
- Language: TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Authentication: iron-session
- Image processing: sharp
- Storage: Oracle Cloud Infrastructure (OCI) Object Storage
