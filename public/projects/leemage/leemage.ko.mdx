## 프로젝트 소개

GitHub : [https://github.com/leey00nsu/leemage](https://github.com/leey00nsu/leemage)

Oracle Cloud Infrastructure(OCI) Object Storage를 활용한 **셀프 호스팅 이미지 관리 플랫폼**입니다.  
Cloudinary와 유사한 기능을 직접 구현하여, 프로젝트 단위로 이미지를 효율적으로 관리하고 개발자 친화적인 API를 제공합니다.

## 주요 기능

- OCI Object Storage 기반 이미지 업로드 및 관리
- 프로젝트 단위 이미지 관리 및 API 제공
- 반응형 대시보드와 직관적 UI
- FSD 아키텍처 기반 확장성 확보

## 기술 스택

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: PostgreSQL, Prisma, Next.js API Routes
- Infra: Oracle Cloud Infrastructure (Object Storage)
