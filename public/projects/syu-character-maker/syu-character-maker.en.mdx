## Project Introduction

GitHub : [https://github.com/leey00nsu/syu-character-maker](https://github.com/leey00nsu/syu-character-maker)

This is a web application that lets you customize Sahmyook University mascots, Suya and Suho.  
Students can freely decorate the characters and save the results as images or share them with classmates.

## Key Features

- React-konva canvas-based decorating tools (pen, stickers, etc.)
- Download the result as an image and upload/share
- State management and optimization using Zustand + TanStack Query

## Tech Stack

- Frontend: React, TypeScript, Tailwind CSS, Zustand, React-konva
- Backend: NestJS, PostgreSQL, Prisma, Redis
