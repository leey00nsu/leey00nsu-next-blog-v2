## Project Overview

Github 주소 : [https://github.com/leey00nsu/syu-character-maker](https://github.com/leey00nsu/syu-character-maker)

Create Your Own Suya and Suho lets you customize Suya and Suho, the mascots of Sahmyook University.
You can upload your customized Suya and Suho and share them with fellow students.

## Key Features

- Use a pen tool and decoration tools on the Suya/Suho canvas to customize them.
- Download the result as an image or upload it to share with fellow students.

## Tech Stack

### Frontend

- Framework: React
- Language: TypeScript
- Styling/UI: Tailwind CSS
- State management: React Query (TanStack Query), Zustand
- Canvas: React-konva

### Backend

- Framework: NestJS
- Language: TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Authentication: Redis
