## 프로젝트 소개

GitHub : [https://github.com/leey00nsu/syu-character-maker](https://github.com/leey00nsu/syu-character-maker)

삼육대학교 마스코트 **수야/수호를 꾸밀 수 있는 웹 프로그램**입니다.  
학생들이 자유롭게 캐릭터를 꾸미고 결과를 이미지로 저장하거나 학우들과 공유할 수 있도록 했습니다.

## 주요 기능

- React-konva 캔버스 기반 꾸미기 도구(펜, 스티커 등)
- 결과물 이미지 다운로드 및 업로드 공유
- Zustand + TanStack Query 기반 상태 관리 및 최적화

## 기술 스택

- Frontend: React, TypeScript, Tailwind CSS, Zustand, React-konva
- Backend: NestJS, PostgreSQL, Prisma, Redis
