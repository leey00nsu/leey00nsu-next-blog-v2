## 프로젝트 소개

Github 주소 : [https://github.com/leey00nsu/syu-character-maker](https://github.com/leey00nsu/syu-character-maker)

나만의 수야,수호 만들기는 삼육대학교 마스코트인 수야, 수호를 꾸밀 수 있는 프로그램입니다.
자유롭게 꾸민 수야, 수호를 업로드하여 학우들과 공유할 수 있습니다.

## 주요 기능

- 수야,수호를 배경으로 펜 툴과 꾸미기 도구를 사용하여 수야, 수호를 꾸밀 수 있습니다.
- 결과를 이미지로 다운로드 받거나, 업로드하여 학우들과 공유할 수 있습니다.

## 기술 스택

### Frontend

- 프레임워크: React
- 언어: TypeScript
- 스타일/UI: Tailwind CSS
- 상태 관리: React Query (TanStack Query), Zustand
- 캔버스: React-konva

### Backend

- 프레임워크: NestJS
- 언어: TypeScript
- 데이터베이스: PostgreSQL
- ORM: Prisma
- 인증: Redis
